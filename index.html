GitHub Flavored Markdown preview plugin

Overview

Sublime Text 2/3 plugin for GitHub Flavored Markdown.

Installation

Sublime Text 2

  1. Install the Sublime Package Control package: http://wbond.net/sublime_packages/package_control/installation
  2. Use Package Control to install this package (GitHubMarkdownPreview)

Sublime Text 3

Download, unzip, and put the contents in <Sublime Text data directory>/Packages/GitHubMarkdownPreview.

Key bindings

"ctrl+shift+g"
